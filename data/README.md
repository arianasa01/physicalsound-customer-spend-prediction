# Data

This project uses two small coursework datasets provided for the Business Analytics module.

- `data/raw/order_july24.csv`: 2,000 sampled July 2024 orders from the fictional PhysicalSound e-commerce website.
- `data/raw/new_customer24.csv`: 20 new customer records used for prediction.

The data does not contain names, emails, addresses, or direct customer identifiers. The `order` field in the new customer file is only a row reference.
